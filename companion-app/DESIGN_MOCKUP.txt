DESIGN PHILOSOPHY:
══════════════════

"Every pixel earned, no corporate jargon, underground polish."

This interface embodies demoscene precision:
  • Dense information displays (no wasted space)
  • Retro terminal aesthetic (CRT GameBoy vibes)
  • Functional glyphs over decorative emojis
  • Monospace typography throughout
  • 4-shade green palette (hardware constraint)
  • Procedural chart rendering (canvas precision)
  • Keyboard-first interaction model
  • Performance-conscious (minimal DOM, efficient JS)

The design feels like it was coded at 3am in a basement by someone who's
optimized too many demo scrollers. Clinical precision meets underground cool.

Not a corporate dashboard. Not a Material Design clone. A tool for sleep
optimization that respects the user's intelligence and screen real estate.


IMPLEMENTATION NOTES:
═════════════════════

▸ Single HTML file (23KB) with inline CSS and embedded JS
▸ Vanilla JavaScript (no frameworks, no build step)
▸ Canvas API for chart rendering (hardware-accelerated)
▸ IndexedDB for persistent storage (async, non-blocking)
▸ PWA-ready (add-to-home-screen meta tags)
▸ Works offline (once loaded)
▸ iPhone Safari tested (iOS 12+)
▸ Keyboard shortcuts for power users
▸ Test data generator for development

Total payload: ~32KB (HTML + decoder.js)
Load time: <100ms on 3G
Frame budget: 60fps chart rendering
Storage: Unlimited IndexedDB (quota permitting)
